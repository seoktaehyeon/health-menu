## 支持生成随机菜单

- 一日菜单
- 一周菜单 
- 一月菜单（30天）

## 如何使用（码农命令行）

#### 安装
```bash
pip install health-menu
```

#### 一日菜单
```bash
health-menu -d
```

#### 一周菜单
```bash
health-menu -w
```

#### 一月菜单
```bash
health-menu -m
```
